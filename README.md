# Mackazon

This is just a personal project made by a guy that is just learning about JavaScript.

## How to preview

1. Clone the repository `git clone https://github.com/osakitadev/mackazon-project`
2. Open the project folder
3. Run `pnpm install`
4. Run `pnpm dev`
